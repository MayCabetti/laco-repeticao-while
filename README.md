# Laço de Repetição com While

Este trabalho teve como objetivo o aprendizado da utilização do comando `while` para criação de laços de repetição em JavaScript. Foram desenvolvidos exercícios práticos envolvendo validação de dados, controle de fluxo e interação com o usuário por meio de entradas via `prompt`.

---

## 1) Validação de Número Positivo ➕

O programa solicita que o usuário digite um número positivo. Caso o valor informado seja negativo, o sistema continua solicitando a entrada até que um número válido seja fornecido.

---

## 2) Cálculo de Média de Notas 📝

O usuário pode inserir várias notas entre 0 e 10. O programa encerra quando um número negativo é digitado e, ao final, calcula e exibe a média das notas válidas inseridas.

---

## 3) Soma de Valores de Compras 💵

O programa permite que o usuário insira valores de compras sucessivamente. Quando o valor 0 é digitado, o sistema encerra e exibe o total acumulado das compras.

---

## 4) Validação de Nome Completo 🥸

O sistema solicita o primeiro nome e o sobrenome do usuário. Caso algum dos campos esteja vazio, o programa continua solicitando até que ambos sejam preenchidos corretamente.

---

## 5) Simulação de Saque Bancário 🏦

Foi implementada uma simulação de saque com saldo inicial definido. O programa verifica se há saldo suficiente para cada saque solicitado. Caso não haja, solicita um novo valor. O processo se repete até que o saldo seja igual ou inferior a zero.

---

## 💡 Conceitos Praticados

- Estrutura de repetição `while`
- Estruturas condicionais (`if/else`)
- Validação de entrada do usuário
- Controle de fluxo
- Operações matemáticas básicas
